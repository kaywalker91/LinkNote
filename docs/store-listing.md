## 3. 자세한 설명 (Full description, 최대 4000자)

```
LinkNote는 흩어진 웹 링크와 메모를 한곳에 모아 깔끔하게 정리하는 북마크 앱입니다.

■ 무엇이든 빠르게 저장
- 다른 앱에서 '공유'만 누르면 링크가 곧장 LinkNote에 저장됩니다.
- 웹 링크, 유튜브 영상, 아티클, 짧은 메모까지 한 공간에 모읍니다.
- 링크를 붙여넣으면 제목·미리보기를 자동으로 가져옵니다.

■ 내 방식대로 정리
- 태그와 컬렉션으로 주제별 분류.
- 즐겨찾기로 자주 보는 링크를 위로.
- 강력한 검색으로 저장한 모든 것을 즉시 찾기 (태그·날짜·즐겨찾기 필터 지원).

■ 읽은 기록 한눈에
- 링크별로 몇 번 열어봤는지, 마지막으로 언제 봤는지 표시됩니다.

■ 가볍게 공유
- 컬렉션을 공개로 전환하면 딥링크 하나로 읽기 전용 공유.
- 받은 사람은 내 데이터를 수정·삭제할 수 없어 안전합니다.

■ 눈이 편한 디자인
- 라이트/다크 모드 모두 정성껏 다듬은 화면.
- 오프라인에서도 저장한 내용을 빠르게 열람.

지금 LinkNote로 흩어진 링크를 깔끔하게 정리해 보세요.
```

## 6. 앱 콘텐츠 (App content) — ⚠️ 사용자 설문 응답

### 6.1 개인정보처리방침
- 문서: `docs/privacy-policy.md`
- ⚠️ 공개 URL로 게시 후 Console에 URL 입력 필요 (GitHub Pages 등).

### 6.2 데이터 보안 양식 (Data safety) — 실제 수집 기준

| 데이터 유형 | 수집 여부 | 목적 | 비고 |
|---|---|---|---|
| 이메일 주소 | 예 | 계정 관리 | Supabase 인증 |
| 사용자 생성 콘텐츠(링크·메모) | 예 | 앱 기능 | Supabase 저장 |
| 앱 활동(화면 조회) | 예 | 분석 | Firebase Analytics |
| 크래시 로그·진단 | 예 | 분석/성능 | Firebase Crashlytics (디버그 제외) |
| 위치·연락처·결제·사진 | 아니오 | — | 미수집 |
| 광고 식별자 | 아니오 | — | 광고 없음 |

- 전송 중 암호화: 예 (HTTPS)
- 사용자가 데이터 삭제 요청 가능: 예 (앱 내 삭제 + 문의처)

### 6.3 콘텐츠 등급 (Content rating) — ⚠️ 설문 응답
- 권장 응답 방향: 폭력/성/도박/약물 **없음** → 전체 이용가 예상.
- 사용자 생성 콘텐츠(공개 공유) 항목은 **있음**으로 응답 (공개 컬렉션 기능).

### 6.4 광고
- 광고 포함: **아니오**

### 6.5 대상 연령층
- 만 13세 이상 일반 (성인 대상 콘텐츠 없음).
