- `buildHighlightedSpans({required String text, required String query})` 헬퍼.
- PR #29 의 `LinkListTile._buildHighlightedSpans` 로직을 추출. case-insensitive lower-cased `indexOf` 반복, 매칭 토큰만 `backgroundColor: AppColors.forestSoft` + `color: AppColors.forestInk` + `fontWeight: w600`.
