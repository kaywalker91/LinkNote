## 트러블슈팅 기록

### Google Fonts 비동기 throw

`google_fonts 6.3.3` 의 폰트 URL 해시가 stale → 첫 시도 fonts.gstatic.com 404. `allowRuntimeFetching = false` 로 전환 시 "font not bundled" Exception. 두 경우 모두 비동기 future 가 unhandled 로 끝나면서 flutter_test 의 per-test zone 이 잡아 모든 골든을 실패 처리. 시도한 우회 6가지(`FlutterError.onError` 오버라이드 / `runZonedGuarded` / `pumpBeforeTest` 의 `GoogleFonts.pendingFonts().catchError`) 모두 flutter_test 가 매 테스트마다 핸들러를 재설정해 무력화. 결국 폰트 번들이 유일한 root-cause fix.

### Fraunces 정적 weight 부족

undercasetype/Fraunces v1.0 정적 set 에 `Fraunces-Medium.ttf` 없음(9pt/72pt/144pt 광학 사이즈 변형만). 해결: `_findFamilyWithVariantAssetPath` 가 파일명 endsWith 매칭만 수행하므로 `Fraunces72pt-Regular.ttf` 를 `Fraunces-Medium.ttf` 로 리네임 → google_fonts 가 weight 500 normal 요청 시 매칭. 시각적으로 medium weight 가 아닌 regular 가 렌더되지만 baseline 일관성에는 문제 없음.
